# Demonstração em Streamlit da Minha API de Forecast

Esta aplicação foi desenvolvida em Streamlit para disponibilizar uma interface simples e interativa de acesso à minha API de previsão de séries temporais (forecast).

Por meio desta demonstração, é possível enviar dados, visualizar previsões e analisar resultados de forma prática e intuitiva.

## Repositório da API: <https://github.com/Pedro101520/Time_Series_Forecast_API/edit/main/README.md>

## Tecnologias Utilizadas:
- Streamlit
- Python
- Pandas
- Matplotlib

## Objetivo
Facilitar o uso da API de forecast através de uma interface, permitindo que usuários testem funcionalidades sem necessidade de chamadas diretas via código ou ferramentas como Postman.
